Lets create js function selectInvestigator(). It will call server getInvestigators method that return list of structures investigator: {name, image1, image2, description}.
Then display a square popup. It will contain two divs. Left div contains investigator names one by one in a row. While mouse over the name, right div will display first and second image and description below the images. Upon clicking the name investigator is considered as selected and popup closed. Submit selected investigator to the server.
